### 🔍 About Me
Soy una desarrolladora de software con sólida experiencia en el ciclo de vida completo de aplicaciones web, ahora enfocada en aplicar ese "know-how" técnico al mundo de la **Ciencia de Datos**.

- 🏗️ **Background:** Años trabajando con PHP (Yii2, Laravel), gestionando servidores y bases de datos relacionales.
- 🛠️ **Full-Stack Mindset:** Complemento mis modelos de datos con herramientas de infraestructura como **Docker** y **Redis** para asegurar un flujo de trabajo profesional.
